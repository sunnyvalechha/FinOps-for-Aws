What is FinOps?
FinOps is a collaboration between finance team and DevOps team to understand and reduce existing cloud cost.

FinOps includes:
* Cloud usage & resources
* Cloud budgeting ex., let's say org budge is 1 million $ so which team utilizing x amount of budget
* Future budget / Forecasting
* Cloud cost optimization - follow best practises and share with other teams (devops/platform/infra).
* Create reporting & dashboards

Tools used in FinOps culture:

* Finout
* Cloudability (recomended)
* Harness
* Cloud Zero
* Aws cost explorer
